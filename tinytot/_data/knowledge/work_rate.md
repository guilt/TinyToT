# Work Rate and Rate Problems

## Rate and Work Problems

Work rate formula: If N workers complete a job in T hours, then 1 worker completes it in N*T hours. Total work = workers × time. For N workers in T hours: rate = 1/(N*T) per hour.

If 3 workers finish a job in 6 hours, the job takes 18 worker-hours total. With 9 workers it takes 18/9 = 2 hours.

Distance equals rate times time: distance = speed × time. Total distance for multiple legs equals the sum of each leg's distance.

Percentage profit: selling price = cost × (1 + profit%/100). A 30% profit on $50 cost gives $50 × 1.30 = $65.

Combined counting: Alice has 8 apples. Bob has half as many as Alice, so Bob has 4 apples. Carol has 3 more than Bob, so Carol has 7 apples. Total = 8 + 4 + 7 = 19.
